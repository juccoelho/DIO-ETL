# Projeto ETL Simples com Python
Este projeto tem como objetivo demonstrar o fluxo ETL (Extração, Transformação e Carregamento) utilizando apenas Python, sem dependência de API ou arquivos externos.

# Etapas do ETL
- Extração: criação de uma lista de usuários fictícios no código
- Transformação: geração de mensagens personalizadas
- Carregamento: exibição das mensagens no terminal
